What is this?

EmailLint is a lint program designed to ensure HTML emails are created using techniques that are available in all major email clients. Certain (excellent) services allow you to generate screenshots to see if your emails work, but nothing pro-actively tells you what you might wrong (at least, not that I could find).

This program was built out of an actual need to validate the compatability a whole range of emails. Manualy verifying and fixing so many custom-constructed emails is time-consuming and boring. This is more fun.
