Application Programming Interface (API)

Coming soon... EmailLint is under active (current) development, so expect updates here. For now, here's some basics:
Request (basic usage) 
curl http://emaillint.com/check
    -d "data=<p>Hello, World! <img src='cid:a.png'/></p>"
    -d "clients[]:GMail"
    -d "clients[]:Yahoo"
    -d "ignored[]:head_style"

    Note: to test all clients, don't send any client[] params
Response 
{
  "success":false,
  "ran":true,
  "errors":[{
    "category":"img",
    "id":"img_alt",
    "name":"Images should have \"alt\" attribute",
    "extended":"In the case where images are not loaded, alt text should be provided to assist comprehension and screen readers",
    "failed":1
  },{
    "category":"img",
    "id":"img_size",
    "name":"Images should have height/width specified",
    "extended":"In the case where images are not loaded, having a height and width specified helps the document render as intended",
    "failed":1
  }]
}
Warning! EmailLint actively throttles to prevent abuse. If you start getting 403 errors, wait a little while before trying again. This is a free service, play nice.
